# Cobbler

Cobbler is a powerful and flexible managing tool for Linux systems. 
It centralises and automates the process of keeping systems up-to-date.
The main use case for Cobbler is in small Raspberry Pi clusters, 
where it simplifies the maintenance of multiple devices.

# Components

Cobbler consists of several key components that work together to provide its functionality:
- **Cobbler Daemon**: A background service that manages system configurations and actions.
- **Cobbler CLI**: A user-friendly command-line interface for managing Cobbler daemon instances across the cluster.
- **Cobbler REST**: A RESTful API for programmatic management of Cobbler daemon instances.
- **Cobbler Web**: A web-based interface for managing Cobbler daemon instances across the cluster.
